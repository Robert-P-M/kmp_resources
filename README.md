# KMP Resources Plugin

<p><b>KMP Resources</b> is an advanced Table Editor and Refactoring Tool for Compose Multiplatform projects.</p>

<p><b>Features:</b></p>
<ul>
  <li><b>Table Editor:</b> View, filter, and edit all your strings, plurals, and string-arrays in a clean UI.</li>
  <li><b>Safe Key Renaming:</b> Rename a resource key and instantly refactor all Kotlin usages (<code>Res.string.*</code>) across your module.</li>
  <li><b>Format Argument Linter:</b> Real-time IDE inspections to prevent missing or mismatched format arguments (like <code>%1$s</code>) in your Compose Multiplatform code.</li>
  <li><b>Quick Documentation:</b> Hover over any KMP resource in your code to see its actual value, type, and navigate directly to translations.</li>
  <li><b>Gradle Sync:</b> Seamlessly triggers Compose resource generation in the background.</li>
</ul>
