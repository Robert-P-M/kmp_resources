Erstellt eine fehlende Compose Multiplatform Ressource direkt aus einer Kotlin-Referenz.

Wenn du auf eine Ressource verweist, die noch nicht existiert (z. B. Res.string.mein_neuer_key), kannst du mit diesem Quick Fix das zugehörige XML-Tag on-the-fly generieren lassen.

Der Quick Fix fragt dich nach dem Standardwert und fügt die neue Ressource sicher in die strings.xml deines aktuellen Moduls ein. Anschließend wird ein Gradle-Sync im Hintergrund gestartet, um die benötigten Kotlin-Klassen zu generieren, was für einen flüssigen Workflow sorgt.
