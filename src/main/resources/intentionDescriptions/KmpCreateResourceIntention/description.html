Creates a missing Compose Multiplatform resource directly from a Kotlin reference.

When you type a reference to a resource that doesn't exist yet (e.g., Res.string.my_new_key), this quick fix allows you to generate the corresponding XML tag on the fly.

The quick fix prompts you for the default value and safely appends the new resource to the strings.xml of your current module. Finally, it triggers a background Gradle Sync to generate the required Kotlin accessors, ensuring a seamless workflow.
